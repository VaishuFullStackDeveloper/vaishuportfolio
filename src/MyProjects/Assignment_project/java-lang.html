Even or Odd

public class EvenOddChecker { public static void main(String[] args) { int number = 7; if (number % 2 == 0) { System.out.println(number + " is even."); } else { System.out.println(number + " is odd."); } } }

Input

Number: 7

Output

7 is odd.

Largest of Two Numbers

public class LargestNumber { public static void main(String[] args) { int a = 10, b = 25; if (a > b) { System.out.println(a + " is larger."); } else { System.out.println(b + " is larger."); } } }

Input

a = 10, b = 25

Output

25 is larger.
